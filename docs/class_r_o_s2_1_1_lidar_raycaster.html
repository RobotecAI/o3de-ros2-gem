Perform raycast against the current scene.

Parameters
startStarting point of rays. This is a simplification since there can be multiple starting points in real sensors.
directionsDirections in which to shoot rays. These should be generated from Lidar configuration.
distanceMaximum distance for ray-casting.
ignoreLayerShould a specified collision layer be ignored
ignoredLayerIndexIndex of collision layer to be ignored
Returns
Hits of raycast. The returned vector size can be anything between zero and size of directions. No hits further than distance will be reported.

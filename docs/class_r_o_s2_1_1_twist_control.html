Detailed Description

Specialization of RobotControl for Twist messages. Twist control moves robot through supplying linear and angular velocities. The control message can be broadcast to EBus system or directly applied to a selected body. This class is used through ROS2RobotControlComponent.

Note
How these velocities are reached might depend on a particular robot drive.

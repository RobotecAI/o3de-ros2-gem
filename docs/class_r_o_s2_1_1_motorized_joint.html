Detailed Description

Helper that allows us to design a joint with a 'motor' It can affect the scene with two APIS: TransformBus and RigidBodyBus TransformBus mode, called AnimationMode changes local transform. In this mode, you cannot have a rigid body controller enabled. It is intended to use in a simple scenario. With RigidBodyBus it applies forces and torque according to PID control.

Note
This class is already used through ROS2FrameComponent.
